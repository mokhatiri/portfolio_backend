# mokhatiri.github.io

Personal portfolio website for Mohammed Khatiri - Computer Science Student specializing in AI and Big Data.

## 🌐 Live Site

Visit: [https://mokhatiri.github.io](https://mokhatiri.github.io)

## ✨ Features

- **Modern Dark Theme** - Sleek, professional design with smooth animations
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile
- **Dynamic GitHub Repos** - Automatically fetches and displays repositories from GitHub API
- **Filter Projects** - Filter repositories by programming language
- **Downloadable CV** - Easy access to download resume
- **Contact Form** - mailto integration for easy communication
- **Smooth Animations** - Typing effect, scroll reveals, and hover effects

## 🛠️ Technologies Used

- HTML5
- CSS3 (CSS Variables, Grid, Flexbox)
- Vanilla JavaScript
- GitHub REST API
- Font Awesome Icons
- Google Fonts (Inter, Fira Code)

## 📁 Project Structure

```
mokhatiri.github.io/
├── index.html
├── README.md
└── assets/
    ├── css/
    │   └── style.css
    ├── js/
    │   └── main.js
    ├── images/
    └── CV_MOHAMMED_KHATIRI_english.pdf
```

## 🚀 Setup

1. Clone the repository
2. Add your CV file: `assets/CV_MOHAMMED_KHATIRI_english.pdf`
3. (Optional) Add a profile image to `assets/images/`
4. Deploy to GitHub Pages

## 📝 Customization

- Edit `index.html` for content changes
- Modify `assets/css/style.css` for styling
- Update `assets/js/main.js` for functionality changes
